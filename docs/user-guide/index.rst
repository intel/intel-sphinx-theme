User guide
##########

This project is focused on providing a comprehensive guide for users to effectively utilize an Intel themed version of the PyData Sphinx Theme.
The theme incorporates elements from the Intel Brand to deliver a visually appealing and consistent user experience,
making it an ideal choice for documentation and web projects that require a cohesive design with Intel branding.

This theme inherits from the `PyData Sphinx Theme version 0.15.4 <https://pydata-sphinx-theme.readthedocs.io/en/v0.15.4/>`_.

There are a number of options for configuring your site's look and feel.
All configuration options are passed with the `html_theme_options` variable in your `conf.py` file.

Intel® Geti™ Documentation
~~~~~~~~~

`The Intel® Geti™ <https://docs.geti.intel.com/>`_ 
platform enables enterprise teams to rapidly build computer vision AI models.
Through an intuitive graphical interface, users add image or video data, make annotations, train,
retrain, export, and optimize AI models for deployment. Equipped with state-of-the-art technology
such as active learning, task chaining, and smart annotations, the Intel® Geti™ platform reduces
labor-intensive tasks, enables collaborative model development, and speeds up model creation.
