Contributor Guide
#################

These pages contain information about the community that leads, supports, and develops this theme, 
including how you can contribute to the project. Contributions are very welcome! In order to get 
started, you can install the development version, build the example docs, and develop the css/js of 
the theme. More details on these tasks can be found in the contributing section of the documentation.
